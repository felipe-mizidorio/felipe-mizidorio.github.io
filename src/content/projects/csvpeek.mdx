## About the project

CSVPeek is a Python library designed to provide a quick and comprehensive peek into CSV files. It generates a report that includes an exploratory data analysis (EDA)
of the file's contents, helping users understand the structure, distribution, and key characteristics of their data with minimal effort.

### Stack used

- **Pandas** for data manipulation and analysis
- **Matplotlib** and **Seaborn** for data visualization
- **Python** for the core logic of the library and report generation
- **Poetry** for dependency management and packaging of the library
