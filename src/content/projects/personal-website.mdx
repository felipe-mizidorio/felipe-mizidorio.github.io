## About the project

This is my personal website, where I centralize papers, projects and research.

### Stack used

The website was built with a modern stack and is fully static:

- **Astro 6** as the main framework
- **React** for interactive components
- **Tailwind CSS v4** for styling
- **MDX** for content
- **GitHub Pages** to host with automatic deploy using GitHub Actions
